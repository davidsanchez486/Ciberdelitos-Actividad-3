## Estructura del repositorio

- **00_overview/**  
  Introducción general al concepto de Prompt Injection, definiciones clave y marcos de referencia iniciales.

- **01_theory/**  
  Fundamentos teóricos, taxonomías y modelos formales relacionados con los ataques de Prompt Injection en LLMs.

- **02_cases/**  
  Casos reales y estudios documentados de ataques mediante Prompt Injection en sistemas y aplicaciones reales.

- **03_techniques/**  
  Descripción de las técnicas y vectores de ataque utilizados para explotar vulnerabilidades de Prompt Injection.

- **04_defenses/**  
  Contramedidas, estrategias de mitigación y enfoques defensivos frente a ataques de Prompt Injection.
